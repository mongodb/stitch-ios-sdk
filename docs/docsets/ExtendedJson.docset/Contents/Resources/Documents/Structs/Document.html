Subscript
  • subscript(_:)

    Accesses the value associated with the given key for reading and writing, like a Dictionary. Document keeps the order of entry while iterating over itskey-value pair. Writing nil removes the stored value from the document and takes O(n), all other read/write action take O(1). If you wish to set a MongoDB value to null, set the value to NSNull.
